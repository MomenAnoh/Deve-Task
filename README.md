# Laravel Products CRUD

This project is a simple CRUD application for managing products built with **Laravel**.

## Features
- Create Product
- Read Product with filtering
- Update Product
- Delete Product

## Requirements
- PHP 8.1+
- Composer
- Laravel 10
- MySQL or SQLite
